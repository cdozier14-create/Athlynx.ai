# TIMEOUT PROCEDURES

## WHEN TO CALL TIMEOUT

1. **System Critical Error** - Stop everything, fix immediately
2. **Team Burnout** - Mandatory rest period
3. **Strategy Pivot** - Major direction change needed
4. **External Crisis** - Market or world event impact
5. **Celebration** - Major milestone achieved

## TIMEOUT PROTOCOL

1. Announce timeout to team
2. Assess situation
3. Develop response plan
4. Communicate changes
5. Resume play
